# Product Guide

## Vision
Quant Unified Client 是一个集成了策略开发、回测验证与实盘交易的统一量化平台。旨在为量化交易者提供一个高性能、可扩展且界面友好的全栈解决方案，将复杂的交易逻辑与可视化的监控管理完美融合。

## Target Audience
*   **量化交易开发者**：需要灵活的框架来编写、测试和部署自定义交易策略。
*   **个人投资者**：希望通过自动化工具管理加密货币资产，实现稳健增值的用户。

## Core Goals
1.  **统一化管理**：在一个 Monorepo 中管理所有策略（网格、择时、做市）与核心库，降低维护成本。
2.  **高性能执行**：利用 Python 异步特性（asyncio/aiohttp）和 CCXT 构建低延迟的实盘交易核心。
3.  **可视化监控**：通过 Qronos 前端平台提供直观的资产曲线、订单状态和策略运行看板。
4.  **无缝产研流程**：打通“策略研发 -> 回测验证 -> 实盘部署”的全链路流程。

## Key Features
*   **多策略引擎**：内置支持网格策略（Grid）、择时策略（Timing）和做市策略（Market Making）。
*   **Qronos 管理台**：基于 Web 的图形化界面，支持策略参数配置、实时日志查看和手动干预。
*   **通用核心库 (Firm)**：标准化的行情获取、订单执行和风控模块，供所有策略复用。
*   **本地化回测**：基于历史数据的快速回测引擎，支持生成可视化报表（HTML/PNG）。
